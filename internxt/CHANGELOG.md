# Changelog

## 1.0.0 (2025-11-23)

### Added
- Initial release
- WebDAV server integration for Internxt cloud storage using the official internxt/webdav Docker image
- Two-factor authentication (2FA) support with both temporary codes and OTP tokens
- Configurable WebDAV port (default: 3005)
- Configurable protocol (HTTP/HTTPS, default: HTTPS)
- Internxt account integration with credentials management
- Comprehensive documentation with usage examples
- Detailed troubleshooting guide
