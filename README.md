# Internxt WebDAV Add-on for Home Assistant

Home Assistant add-on repository for Internxt WebDAV server.

## Installation

1. In Home Assistant, navigate to **Settings** → **Add-ons** → **Add-on Store**
2. Click the three dots in the top right corner and select **Repositories**
3. Add this repository URL: `https://github.com/cgfm/internxt-addon`
4. Find "Internxt WebDAV" in the add-on store and click **Install**

## Add-ons in this repository

### Internxt WebDAV

Access your Internxt cloud storage through WebDAV protocol.

See the [add-on documentation](internxt/README.md) for more details.
